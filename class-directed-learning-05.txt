 | 0 |           0 |                            0 | 2^0 = 1     |
 | 1 |          01 |                            1 | 2^1 = 2     |
 | 2 |          10 |                            1 | 2^1 = 2     |
 | 3 |          11 |                            2 | 2^2 = 4     |
 |   |             |                              |            |
** In terms of the Binomial Theorem\rsquo{}s (1 + x)^n reduced mod 2
*** Expand and Reduce
    (1 + x)^2 = 1 + 2x + x^2 (mod 2 = 1 + 0x + x^2 = 1 + x^2)
*** Try with an odd power
    (1 + x)^3 = (1 + 2x + x^2) + (x + 2x^2 + x^3) = 1 + 3x + 3x^2 + x^3, mod 2 ==> 1 + 1x + 1x^2 + x^3
*** Better with even
    (1 + x)^4 = 1 + 4x + 6x^2 + 4x^3 + x^4,  mod 2 ==> 1 + x^4
*** Or decomposed into power of 2 powers
    (1 + x)^11 = (1 + x)^(8 + 2 + 1) ==> (1 + x)^8 (1 + x)^2 (1 + x)^1 mod 2 = (1 + x^8)(1 + x^2)(1 + x^1)
